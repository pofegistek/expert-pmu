# Admin Guide — Expert PMU

Этот файл — шпаргалка по управлению содержимым Mini App через Telegram-бот.

## Кто может
Только admin с Telegram ID `317647581`.

## Команды

### Мастера
- `/masters` — список всех мастеров. Кнопки: добавить, редактировать, удалить.
- `/addmaster_v2` — пошаговое добавление нового мастера (шаг фото поддерживает загрузку прямо в чат).

### Услуги
- `/services` — список всех услуг. Кнопки: добавить, редактировать, удалить, галерея.
- `/addservice_v2` — пошаговое добавление новой услуги (в конце предложит добавить фото галереи).

### Галереи работ
- В `/services` → карточка услуги → кнопка **🖼 Галерея** — управление общей галереей работ услуги:
  - ➕ Добавить фото — отправить фото в чат боту (по одному, затем /done)
  - 🗑 Удалить фото — выбрать из списка
  - 📋 Список фото — показать все файлы

- В `/masters` → карточка мастера → кнопка **🖼 Фото работ** — управление персональными галереями мастера по услугам:
  - Выбрать услугу → добавить/удалить/смотреть фото

### Фото мастера
- Через `/addmaster_v2` шаг «Фото» — кнопки: 📸 Прислать фото / 🔗 Ввести ссылку / ⏭ Пропустить
- Через `/masters` → ✏️ Фото — кнопки: 📸 Прислать фото / 🔗 Ввести ссылку

### Прочее
- `/addworks` — старая команда добавления фото работ (совместимость, работает по-прежнему).
- `/price` — изменить цену услуги.
- `/masterprice` — изменить цену «от» для мастера.
- `/broadcast` — рассылка всем пользователям.

## Галереи — как работает

Данные о фото хранятся в трёх местах:
1. **Файлы** на VPS в `/root/expert-pmu/images/works/` → публикуются на GitHub Pages
2. **БД** (`service_gallery_photos`, `master_service_photos`) — индекс для API
3. **photos.json** в папке каждой галереи — для обратной совместимости

При добавлении фото через бота: файл сохраняется → запись в БД → git push в фоне.

## Архитектура API

- `GET /services-sections` — услуги с полями `has_service_gallery`, `gallery_photo_count`, `master_has_gallery`
- `GET /masters-full` — мастера с полями `has_any_gallery`, `gallery_services`
- `GET /service-gallery/{id}` — список фото галереи услуги
- `GET /master-service-gallery/{master_id}/{service_id}` — список фото мастера для услуги
- `GET /master-galleries/{master_id}` — список галерей мастера
